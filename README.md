# ⏱️ TimeFlow AI | Smart Time & Location Tracker

تطبيق ويب احترافي لإدارة الوقت والمهام والإنتاجية مع تتبع المواقع الجغرافية. يعمل بالكامل محلياً، بدون سيرفر، ويدعم العمل بدون إنترنت.

## ✨ الميزات
- 🗄️ قاعدة بيانات محلية IndexedDB (8 مخازن منفصلة)
- 🔐 نظام مصادقة كامل (تسجيل، دخول، ضيف، تذكرني)
- 📋 إدارة مهام متقدمة (CRUD، فلترة، فرز، أرشفة، نسخ، مهام فرعية)
- ⏲️ مؤقت ذكي مع دعم بومودورو وتتبع الجلسات
- 📍 تتبع مواقع مباشر مع خرائط Leaflet + OSM
- 📊 تحليلات وإحصائيات تفاعلية باستخدام Chart.js
- 🌍 دعم كامل للغة العربية و RTL
- 🌙 وضع ليلي/نهاري مع حفظ التفضيلات
- 📲 PWA كامل مع دعم التثبيت والعمل أوفلاين
- 🔒 أمان: تشفير كلمات المرور، تطهير المدخلات، منع XSS
